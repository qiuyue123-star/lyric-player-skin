# 自定义歌词详情播放器美化插件
## 插件信息
- 插件ID：lyric-player-custom-skin
- 版本：v1.0.0
- 作者：篍约
- 适配软件：MoeKoeMusic

## 功能介绍
1. 内置三款播放器美化主题：星空彩胶 / 信笺歌词 / 梦幻海域
2. 设置弹窗一键切换，点击「永久保存」写入本地存储，重启软件不丢失配置
3. 「恢复默认」一键还原原生播放器样式
4. 弹窗内置插件信息面板，展示作者、版本、创作时间与简介

## 使用方法
1. 将全部文件打包为zip压缩包
2. 打开MoeKoeMusic → 设置 → 插件 → 安装插件，选择zip文件导入
3. 点击插件右上角设置按钮，打开主题选择弹窗
4. 选择喜欢的样式，点击永久保存即可实时生效

## 注意事项
- 插件使用浏览器存储持久化配置，清除软件缓存不会丢失皮肤设置
- 样式仅作用于歌词详情全屏播放器界面，不影响主界面布局
